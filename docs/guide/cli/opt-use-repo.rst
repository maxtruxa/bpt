.. option:: --use-repo <url-or-path>, -r <url-or-path>

    Specify the :term:`URL` or :term:`filepath` to a package repository to use
    for package/dependency lookup. This option can be repeated for multiple
    repositories. The default repository will always be available unless
    :option:`--no-default-repo` is specified.

    Only packages in any expressly-enabled repositories will be available.
