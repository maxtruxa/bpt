.. option:: --toolchain <file-or-id>, -t <file-or-id>

    Specify the :doc:`toolchain </guide/toolchains>` to use. Can be a
    :term:`filepath` to a :ref:`toolchain file <toolchains.file>` or the name of
    a :ref:`built-in toolchain <toolchains.builtin>`. If not provided, |bpt|
    will try to load a :ref:`default toolchain <toolchains.default>`.
